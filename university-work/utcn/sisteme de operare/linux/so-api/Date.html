unsigned char*  getAsBytes ()

getAsBytes

Returns the date as a 4 byte array, with the first byte representing the day, the second byte representing the month and the last two bytes representing the year.

Returns: the date as a 4 byte array.

void  setFromBytes (unsigned char* bytes)
throw(InvalidDateException*)

setFromBytes

Sets all the date's data members using the 4 byte representation received as parameter. The 4 byte array is interpreted as follows: the first byte is thought to represent the day, the second byte is thought to represent the month and the last two bytes are thought to represent the year. If an invalid date results from the conversion, an InvalidDateException is thrown and all of the data members are left unchanged.

The method throws:

  • InvalidDateException* if the converted date is invalid.

Parameters:
bytesa 4 byte array containing the date in the format described above.
