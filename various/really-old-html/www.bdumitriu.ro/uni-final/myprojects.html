Once this module is installed in the Apache web server users can create files with a special extension in which, between specified tags, they can insert Prolog code.     Year:2001
    Nr. of persons:1


  2. Project description: The largest project I have worked on until now is called "Coned Virtual University". Basically, it is a web programming project aimed at creating a virtual space usable by both students and teachers for learining / teaching purposes. A colleague of mine and I have started this project from scratch, under the supervision of the teacher in charge of the laboratory we worked in (distributed systems lab), creating a software architecture for the entire system and then developing the application itself.

    The main features of the system are:
    • static materials presentation (presentation of course specific materials)
    • dynamic material presentation (presentation of class specific materials)
    • conference services (area where topic based message exchange takes place)
    • chat room services (area where real-time general purpose conversation takes place)
    • email services (a means of contact among members of a class ? teacher and students)
    • announcement services (a means for the teacher to announce class related events)
    • task assignment (a means for the teacher to assign tasks to the students)
    • task submission (a means for students to submit solved tasks back to the teacher)
    • task tracking (a means for students to track the status of their submitted task)
    • grading book (area where the teacher can award grades to his students)

    The project can currently be found at this address: http://www.coned.utcluj.ro/ but, unfortunatelly, hardware problems with the computer running the database server prevent if from being usable at this time. I hope these problems will be solved as soon as funds are available for the lab.
    Technologies / tools: JSP, Servlet, JDBC, Java, Tomcat, Microsoft SQL Server
    Year:2001-2002
    Nr. of persons:2
