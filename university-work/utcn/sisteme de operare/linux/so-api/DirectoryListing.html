Detailed Description

This class contains a set of directory entries (i.e. files or (sub)directories) which represent a directory's contents. It allows manipulation of this set of entries.

 DirectoryListing ()

DirectoryListing

Creates an empty directory listing with an initial capacity of 50. This capacity represents the number of entries the internal array can store. If this capacity is reached a new array with a double capacity is allocated and the old one is copied into the new one.
