Detailed Description

This class makes inode addressing transparent. It allows the user not to care whether (s)he's working with direct or indirect addressing.

 FileBlocks (HDD* hdd, Inode* inode)

FileBlocks

Creates a new FileBlocks. hdd is the hard disk to be used for read/write operations (what will be read/written are the blocks used for indirect addressing). inode represents the inode associated with the file whose block addresses you want to read/write. The inode received as parameter is supposed to be valid and is used as such.

Parameters:
hddthe hard disk to be used for read/write operations. @ @param inode the inode of the file whose block addresses you want.

The destructor of the class. This destructor does not free either the hdd or the inode received as parameters in the constructor.

getBlock

Returns the address of the indexth block of data of the file.

The method throws:

  • ArrayIndexOutOfBoundsException* if index is less than 0 (which is rather difficult, since it's an unsigned long) or greater than or equal to include/defs.h::MAX_BLOCKS.
  • HardDiskNotInitializedException* forwarded from hdd.
  • IOException* forwarded from hdd.
  • InvalidBlockNumberException* forwarded from hdd.

Parameters:
indexthe index of the file's block whose address you want.

Returns: the address of the indexth block of data of the file.

void  setBlock (unsigned long index, unsigned long blockAddress)
throw( ArrayIndexOutOfBoundsException*, HardDiskNotInitializedException*, InvalidBlockNumberException*, IOException*)

setBlock

Sets the address of the indexth block of data of the file to blockAddress. Beware that the method does not allocate space for the indirection blocks. It simply assumes that the index you provide is an index in an already allocated & written in the inode structure indirection block.

Remeber to write the inode on the disk after calling this method.

The method throws:

  • ArrayIndexOutOfBoundsException* if index is less than 0 (which is rather difficult, since it's an unsigned long) or greater than or equal to include/defs.h::MAX_BLOCKS.
  • HardDiskNotInitializedException* forwarded from hdd.
  • IOException* forwarded from hdd.
  • InvalidBlockNumberException* forwarded from hdd.

Parameters:
indexthe index of the file's block whose address you want to change.
blockAddressthe block address you want to set the indexth block address to.

getBlocks

Returns the addresses of length blocks of data of the file starting with beginIndex. The array will contain length elements.

The method throws:

  • ArrayIndexOutOfBoundsException* if beginIndex+length is less than 0 (which is rather difficult, since they're both unsigned longs) or greater than or equal to include/defs.h::MAX_BLOCKS.
  • HardDiskNotInitializedException* forwarded from hdd.
  • IOException* forwarded from hdd.
  • InvalidBlockNumberException* forwarded from hdd.

Parameters:
beginIndexthe index of the first block whose address you want.
lengththe number of blocks you want.

Returns: an array containing the addresses of length blocks of data starting with beginIndex.

void  setBlocks (unsigned long beginIndex, unsigned long length, unsigned long* addresses)
throw( ArrayIndexOutOfBoundsException*, HardDiskNotInitializedException*, InvalidBlockNumberException*, IOException*)

setBlocks

Sets the addresses of length blocks of data of the file starting with beginIndex to the values contained in the addresses array. The array should therefore contain at least length valid elements.

Remeber to write the inode on the disk after calling this method.

The method throws:

  • ArrayIndexOutOfBoundsException* if beginIndex+length is less than 0 (which is rather difficult, since they're both unsigned longs) or greater than or equal to include/defs.h::MAX_BLOCKS.
  • HardDiskNotInitializedException* forwarded from hdd.
  • IOException* forwarded from hdd.
  • InvalidBlockNumberException* forwarded from hdd.

Parameters:
beginIndexthe index of the first block whose address you want.
lengththe number of blocks you want.
addressesthe values of the block address you want to set.
