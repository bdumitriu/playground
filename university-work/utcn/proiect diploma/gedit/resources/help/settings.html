Application settings



The applications permit settings refering to the functionaliti of the garbage collector.To modify this settings select the Options/Settings menuitem from the application main menu. The garbage collecter is an incorporated part of the applications which scope is clear the old, allready comited opertions after a specified time. For more details about the garbagecolector fucntionality see "Diploma thesis - GEditor" The user ca set the garbage collector operation mode:

  1. Automatic
  2. Manual

In the first case,automatic operation mode the garbage collector runs periodically. The user can set the duration of this period. In the secon case,manualthe garbage collector will be activated only by user interaction, pushing the Clear button.
