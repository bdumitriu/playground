Detailed Description

This class encapsulates data pertaining to a directory entry (normally, a file or a (sub)directory). It provides a constructor which creates the entry using the information found in an inode and getters for all of the elements.

 DirectoryEntry (Inode* inode, unsigned long inodeNumber)

DirectoryEntry

Creates a new DirectoryEntry and populates its data members using the following values from the Inode object:

  • the file type.
  • the file name (and for this, the file name size...).
  • the file size in bytes.
  • the file size in blocks.
  • the file creation date.
  • the file last access date.
  • the file last update date.

It assumes that inode contains valid information.

Parameters:
inodethe valid inode containing the required information.
inodeNumberthe actual number of this inode.
